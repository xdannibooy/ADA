Repositorio correspondiente a la materia Análisis y Diseño de Algoritmos, cursada en la Escuela Superior de Cómputo del Instituto Politécnico Nacional.

Autor: Daniel Solis Bustos
Estudiante de Ingeniería en Sistemas Computacionales
Escuela Superior de Cómputo – IPN

Este repositorio contiene prácticas, ejercicios, implementaciones y material de apoyo desarrollados a lo largo del curso de Análisis y Diseño de Algoritmos. 
Su finalidad es reforzar la comprensión del diseño, análisis y evaluación de algoritmos mediante el estudio de su comportamiento y eficiencia.

Los temas abordados se basan en el temario oficial de la materia, el cual se encuentra disponible de manera pública en la página oficial de la Escuela Superior de Cómputo. 
Las implementaciones realizadas reflejan los contenidos vistos durante el semestre y el proceso de aprendizaje del autor.

El repositorio incluye análisis de complejidad temporal y espacial, uso de notación asintótica, algoritmos de búsqueda y ordenamiento, soluciones recursivas e iterativas,
así como comparaciones de rendimiento entre distintos algoritmos.
El código está pensado con fines académicos y de estudio, priorizando la claridad y comprensión de los conceptos.

El lenguaje de programación principal utilizado es C y python, apoyándose en compiladores estándar y el uso de Git y GitHub como sistema de control de versiones. 
Este repositorio tiene un propósito estrictamente educativo y documenta el trabajo realizado durante el curso.
